## 📌 Overview

[cite_start]This interactive art-based research project explores the implicit dynamics of collective conformity and peer pressure within social environments[cite: 1, 9, 92]. 

[cite_start]Rather than framing peer pressure as an immediate or linear force, this system models it as a process of **hidden accumulation, repetition, and threshold-based eruption**[cite: 43, 93]. [cite_start]It draws a conceptual parallel to physical phenomena like **Metal Fatigue** or an iron burn on fabric—where continuous, invisible stress builds up silently inside a structure until it triggers an abrupt, irreversible fracture[cite: 109, 110].
